- 🔒 稳定性增强

## 🔄 自动调度机制说明

本次升级后系统具备更强的自动恢复能力：

- 节点 IP 被封锁或异常时自动切换线路  
- 异常线路自动替换  
- 降低人工切换依赖  

---

### 📌 使用建议

若出现短暂无法连接情况：

- ❌ 请勿频繁切换节点  
- ⏳ 等待 1–3 分钟  
- 🔄 执行一次“更新订阅”  

通常即可恢复正常。
